Section 2.2 Comparison Test for integrals

In many occasions the actual value of an improper integral is hard to find. Perhaps, it is because an anti-derivative of the integrand is hard to find. Perhaps, the limit involved is hard to compute. However, sometime all that we care about is whether the improper integral converges or not. If that is the case, it is often much easier to decide the convergence of the integral by comparing it with another simpler improper integral whose convergence is easier to determine.

For simplicity, we assume all functions in the following discussion are integrable on whatever the domain of integration that we talk about.

We begin with an observation: if the graph of \(f\) never dips below the \(x\)-axis on the interval \([a,b]\text{,}\) the area bounded between the graph of \(f\) and the \(x\)-axis cannot be negative (it can be zero though). Hence,

Proposition 2.2.1.

Suppose \(f(x) \ge 0\) on \([a,b]\text{,}\) then \(\int_a^b f(x) dx \ge 0\text{.}\)

Consequently, if \(f \le g\) on \([a,b]\text{,}\) then \(g-f \ge 0\) (which is also integrable) on \([a,b]\) and so

\begin{equation*} 0 \le \int_a^b g(x)-f(x) dx = \int_a^b g(x)dx - \int_a^b f(x)dx. \end{equation*}

That is, \(\int_a^b f(x)dx \le \int_a^b g(x)dx\text{.}\) Note also that if \(f \ge 0\text{,}\) the function \(A(t) = \int_a^t f(x)dx\) is an increasing function of \(t\text{.}\) Therefore, if \(A(t)\) is bounded above on \([a,\infty)\text{,}\) then \(\int_a^{\infty} f(x)dx = \lim_{t\to \infty} \int_a^t f(x)dx\) exists. From this we conclude that

Proposition 2.2.2.

(Comparsion test for improper integrals) Suppose \(0 \le f \le g\) on \([a,\infty)\text{,}\) then

  • if \(\int_a^\infty g(x)dx\) converges, so does \(\int_a^{\infty} f(x) dx\text{.}\) Moreover, \(\int_a^{\infty} f(x)dx \le \int_a^\infty g(x)dx\text{.}\)

  • if \(\int_a^{\infty} f(x)dx\) diverges to \(+\infty\text{,}\) then so does \(\int_a^\infty g(x)dx\text{.}\)

The same results hold for improper integrals with bounded domain of integration (see Exercise 2.3.1).

Let us put the comparison test into action.

Example 2.2.3.

Consider the proper integral \(\int_1^{\infty} \frac{1}{x^3 + 2} dx\text{.}\) The indefinite integral \(\int \frac{1}{x^3+2}dx\) is elementary but not that easy to find by hand. But it is easy to determine the convergence of the improper integral. Since \(0 \lt \frac{1}{x^3+2} \lt \frac{1}{x^3}\) on \([1,\infty)\text{.}\) Since \(\int_1^{\infty} \frac{1}{x^3}dx\) converges to \(1/2\) by Proposition 2.1.3, it follows from Proposition 2.2.2 that
\begin{equation*} \int_1^{\infty} \frac{1}{x^3+2}dx \le \int_1^{\infty} \frac{1}{x^3}dx = \frac{1}{2}. \end{equation*}

Exercise 2.2.4.

Show that

\begin{equation*} \int_1^{\infty} \frac{\cos^2(x)}{1+x^2} dx \le \frac{\pi}{4} \end{equation*}
Solution.

Since \(0 \le \cos^2(x) \le 1\text{,}\) \(0 \le \frac{\cos^2(x)}{1+x^2} \le \frac{1}{1+x^2}\text{,}\) it follows from the comparison test for integrals that

\begin{align*} 0 & \le \int_1^{t} \frac{\cos^2(x)}{1+x^2}dx \le \int_1^{t} \frac{1}{1+x^2}dx \\ & = \arctan(t)-\arctan(1) \to \frac{\pi}{4} \quad \text{as}\ t \to \infty. \end{align*}

This establishes the inequality \(\int_1^{\infty} \frac{\cos^2(x)}{1+x^2} dx \le \frac{\pi}{4}\text{.}\)
